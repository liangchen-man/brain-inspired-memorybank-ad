# 第一周任务逐题回答附录

本附录逐条回答第一周任务清单中的核心问题，便于最终报告直接引用。

## SPADE

**1. SPADE 保存的 memory bank 里面到底是什么？**

SPADE 保存的是正常图像的深度特征表示。本项目教学版保存每张正常图像的全局特征和空间特征图。它相当于保留一组“正常参考样本”。

**2. 测试图像如何和正常图像进行比较？**

测试图像先提取全局特征，检索最相似的正常图像；再比较测试图像与这些正常参考图像的局部空间特征差异，差异越大越异常。

**3. 为什么 SPADE 方法直观有效？**

它模仿了“拿当前样本和正常样本对照”的直觉。如果测试图像的某个区域无法被相似正常样本解释，就可能是异常。

**4. 为什么 SPADE 的内存和搜索开销可能较大？**

因为它保存的是正常样本级特征和空间特征图。正常样本越多，memory bank 越大，检索和局部比较越慢。

**5. 它对真实工业现场有什么局限？**

真实现场会有光照、遮挡、摆放变化和工人干扰。若正常参考库没有覆盖这些变化，SPADE 容易误报。

## PaDiM

**1. PaDiM 的 memory bank 和 SPADE 有什么不同？**

SPADE 保存正常样本特征；PaDiM 不直接保存所有样本，而是在每个空间位置保存正常特征的均值和协方差，是统计式记忆。

**2. 均值代表什么？**

均值代表某个空间位置上正常特征的中心模式，也就是该位置通常应该长什么样。

**3. 协方差代表什么？**

协方差描述正常特征在不同维度上的波动范围和相关性。它表示正常变化允许有多大。

**4. 为什么使用马氏距离，而不是简单欧氏距离？**

马氏距离会考虑不同特征维度的方差和相关性。如果某个方向本来波动很大，偏离一点不一定异常；如果某个方向本来很稳定，偏离一点就更可疑。

**5. PaDiM 是否适合位置变化很大的工业图像？**

不太适合。PaDiM 假设同一空间位置的正常模式相对稳定，如果工件位置大幅移动，位置统计记忆会失效。

## PatchCore

**1. PatchCore 为什么叫 PatchCore？**

它以 patch 级特征为核心，并通过 coreset 选择最有代表性的正常 patch，因此可以理解为 patch-level core memory。

**2. coreset 解决了什么问题？**

coreset 用较少的代表性 patch 近似完整正常 patch 集合，减少 memory bank 大小和最近邻搜索成本。

**3. PatchCore 为什么通常比 SPADE 更高效？**

PatchCore 不需要保存所有正常图像的完整空间特征图，而是保留压缩后的代表性 patch。memory bank 更小，检索更轻。

**4. PatchCore 和 PaDiM 相比，哪一个保留了更多正常样本细节？**

PatchCore 保留更多局部样本细节，因为它直接保存正常 patch 特征；PaDiM 将样本压缩为位置统计分布。

**5. PatchCore 是否已经解决了真实工业现场的全部问题？**

没有。它主要处理静态图像中的局部视觉异常，不能直接理解视频流程、工位动作顺序和长期场景漂移。

## AnomalyDINO

**1. AnomalyDINO 是否仍然属于 memory bank 范式？**

属于。本项目教学版使用 DINOv2 patch tokens 构建正常 patch token memory bank，测试 token 通过最近邻距离得到异常分数。

**2. DINO 特征为什么可能更适合异常检测？**

DINOv2 是自监督视觉表征，可能更关注物体结构、局部一致性和跨场景泛化，而不是只关注分类标签。

**3. 它和 PatchCore 的主要区别是什么？**

PatchCore 本项目使用 ResNet18 patch 特征；AnomalyDINO 使用 DINOv2/ViT patch token。两者都用 patch memory，但特征来源不同。

**4. 它对少样本工业异常检测有什么意义？**

强预训练特征可以减少对大量工业缺陷样本的依赖，使少量正常样本也能构建较有效的正常记忆。

**5. 它是否可以直接用于我们的应用？如果不能，主要障碍是什么？**

不能直接保证可用。真实工位存在视频时序、光照漂移、遮挡、工人操作和生产批次变化，需要结合时间平滑、动态 memory bank 和人工反馈机制。

## 总结回答

第一周实验说明：工业异常检测可以不先学习所有缺陷，而是通过不同形式的 memory bank 建立正常性记忆。第二周工位异常监控则要进一步验证：这些静态正常性记忆在真实现场变化下是否仍然可靠，以及是否需要短期记忆、长期记忆和人工反馈。
